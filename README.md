# 🤖 UX Agent: 장군이

본 저장소는 디자인-개발 정합성 검토 및 UX 리서치 자동화를 위한 AI 에이전트 **'장군이'**의 설정 파일 모음입니다.

## 🛠️ 시작하기
1. 본 리포지토리를 `git clone` 하여 로컬에 저장합니다.
2. Antigravity 또는 IDE의 에이전트 매니저에서 `systemPrompt` 폴더를 참조 경로로 지정합니다.

## 📁 폴더 구조
- `systemPrompt/`: 장군이의 페르소나 및 기술 매뉴얼 (01~04.md)
- `workspace/`: 실제 작업 파일(디자인 JSON, 코드)을 두는 공간

## 🛡️ 장군이의 주요 기능
- Figma 디자인과 퍼블리싱 코드 간의 수치 정합성 자동 검수
- Deep Research를 활용한 자율 UX 트렌드 분석 및 리포트 생성
